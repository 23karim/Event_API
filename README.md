# Event_API
API REST Node.js/Express connectée à une base PostgreSQL (Neon) permettant la gestion complète des événements

---

##  Guide d'utilisation

Pour cloner et lancer cette API, vous devez avoir **Git**, **Node.js** (Version 18.x ou supérieure) et **npm** installés sur votre ordinateur.

###  Lancement via le terminal

Exécutez les commandes suivantes dans votre terminal pour configurer et démarrer le projet :

```bash
# 1. Cloner ce dépôt
$ git clone https://github.com/23karim/Event_API.git

# 2. Accéder au dossier du projet
$ cd Event_API

# 3. Installer les dépendances
$ npm install

# 4. Lancer le serveur en mode développement
$ node server.js
```
Une fois le serveur lancé, l'API sera accessible sur : http://localhost:5000
